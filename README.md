How to install
==============

1. Install node.js

2. Install imagemagick (required for build process to optimize background images)

Ubuntu:
-------

apt-get install imagemagick
apt-get install graphicsmagick

Mac OS X (using Homebrew):
--------------------------

brew install imagemagick
brew install graphicsmagick

Windows & others:
-----------------

http://www.imagemagick.org/script/binary-releases.php

3. Run npm install

4. Run gulp

5. Run node app.js

Useful commands
===============

gulp clean

Clean build directory

gulp

Build the project
